How to use

Simply copy the HTML & CSS from this file and paste it in your web page. The HTML code starts from <h4>MacBook Photo Slider</h4> and ends at the second to the last </div>. Make sure you copy the internal CSS in this file to your web page. The Javascript functions are defined in slideshow.js. Make sure you load slideshow.js and of course jquery.min.js!

Closer Look (click here)

If you look at the source code, you will notice the number "0" in front of some ids, for example id="left0" or onclick="clickSlide(false,'0');". If you want to change the ids to something else, you could change this number to whatever number you like (let's call it "x"), and make sure you call the init('x') function when the document is "ready" like this:

function start(){ 
    init('x'); 
    //more functions...
} 
$(document).ready(start);
Change even more?

If you want to replace the pictures (like iPhone) with other pictures (like Android phone, or tablet), or if you want to make changes to the structure of the code, you should change the scale numbers in CSS; for example max-width: 1000px or height: 625px are defined for a 1000x625 picture.

If you add a new slideshow with arbitrary class name, make sure you add the respective "resize" function in the slideshow.js, to resize it when screen size changes (for a responsive design). The function below shows a "resize" function for a MacBook Image Slider:

for (var i = 0; i < $('.mac').length ; i++){ // resize mac when screen is being resized
    var w = $('.mac').eq(i).width(); // "mac" is the class name of the "div" that contains the MacBook Image Slider
    $('.mac').eq(i).height(0.625 * w); // Mac image ratio is 1000x625, or 1 to 0.625
}

For responsive design one needs to call the following two functions when the page is "ready" (e.g. in the function "start()" mentioned above): 

$(window).resize(function() {resizeScreen();});
$(window).resize();
